
Student Performance Prediction
Overview
Predict students' final performance using historical and behavioural features. Includes data preprocessing, model training, evaluation, and a simple inference API.

Features
Data cleaning and validation
Feature engineering
Model training (Random Forest + baseline models)
Hyperparameter tuning
Evaluation metrics and plots
Flask-based inference endpoint
Tech stack
Python, pandas, numpy, scikit-learn, matplotlib, flask, joblib

Install
pip install -r requirements.txt
